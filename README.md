To run Flask app:
- You need to have Python installed to run the app

1. Open the terminal
2. Run ".\venv\Scripts\activate" to set up the virtual environment
3. Type "cd app" to change the directory
4. Run "pip install -r requirements.txt" to download all requirements
5. Type "python app.py" to run the Flask app
6. Go to http://127.0.0.1:5000 to see it running in your web browser
